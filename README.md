## About The Project


This project uses [GithubAPI](https://docs.github.com/en/rest?apiVersion=2022-11-28) to show lists of user's repositories.


### Project Structure

## Getting Started

### Prerequisites

This is an example of things you need to use the application and how to install them.

* [node.js](https://nodejs.org/en/download/)

### Installation

1. Clone the repo
```sh
git clone https://github.com/reijiren/List_Github_Repos.git
```
2. Change directory to listgithubrepo
```sh
cd .\listgithubrepo\
```
3. Install NPM packages
```sh
npm install
```
4. Build project
```sh
npm run build
```
5. Run the project
```sh
npm run start
```

or run in dev mode

```sh
npm run dev
```
